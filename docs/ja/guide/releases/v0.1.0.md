# v0.1.0 リリースノート

![v0.1.0 release header](/release-header-v0.1.0.svg)

`v0.1.0` は `onizuka-openclaw-autonomous-team-starter` の最初のタグ付きリリースです。前の tag が無いため、このページは `main` に入っている全履歴を初回リリースとしてまとめています。

## スコープ

- 初回リリース: `v0.1.0` までの全履歴
- 出荷面: Python CLI、PowerShell wrapper、tracked `.openclaw` scaffold、英日 VitePress docs、Mattermost plugin bundle、検証 artifact
- companion article: [v0.1.0 公開ウォークスルー](/ja/guide/articles/v0.1.0-launch)

## ハイライト

### Windows 前提の自律 OpenClaw チームスターター

- `src/openclaw_podman_starter/cli.py` の Python CLI と `scripts/` の PowerShell entrypoint により、`init`、`doctor`、`launch`、`status`、`logs`、`stop`、`print-env`、Mattermost 管理、autostart 補助を一通り揃えています。
- 単体起動と multi-instance 起動の両方を持ち、dry-run launch/stop も CI で確認しています。

### agent ごとの state・pod・persona scaffold

- `.openclaw/instances/agent_*/` 以下に agent ごとの `openclaw.json`、`pod.yaml`、workspace を生成します。
- workspace には `AGENTS.md`、`SOUL.md`、`IDENTITY.md`、`USER.md`、`HEARTBEAT.md`、`TOOLS.md`、`BOOTSTRAP.md` が入り、generic container ではなく teammate として立ち上げられます。
- 生成状態のうち sanitized subset を repo に追跡し、manifest と persona 変更をレビュー可能にしています。

### heartbeat-first の Mattermost 会話ラボ

- `mattermost init`、`launch`、`seed`、`smoke`、`lounge enable`、`lounge status`、`lounge run-now` を starter CLI と PowerShell wrapper から扱えます。
- `scripts/mattermost_tools/` には state 取得、post、reaction、public channel 作成の helper を分離して収録しています。
- instance ごとの autonomy interval override と、6 席構成の persona/model scaffold を持っています。

### bilingual docs・CI・GitHub Pages

- `docs/` の VitePress site を英語と日本語で公開します。
- Windows CI では `uv sync`、Python compile、unit test、docs build、CLI help、single-instance init/print-env、multi-instance dry-run launch/stop を回します。
- GitHub Pages から operator guide と release collateral を同じ docs surface で配信します。

### このリリースに含まれる docs と asset

- 検証レポートと Mattermost autonomy QA inventory を `reports/` に同梱しています。
- 実験用の Rokuseki Mattermost branding plugin bundle と UI 検証スクリーンショットを `mattermost-plugins/` と `reports/` に収録しています。

## Tooling と automation

- `pyproject.toml` では starter package を `0.1.0` として公開できる状態に揃えています。
- `.github/workflows/ci.yml` は Python package、docs build、operator 向け dry-run を Windows で保護します。
- `.github/workflows/pages.yml` は VitePress 出力を docs site として配信します。

## 検証

このリリース作業では次のコマンドを実行しました。

```powershell
powershell -ExecutionPolicy Bypass -File D:\Prj\gh-release-notes-skill\scripts\collect-release-context.ps1 -Target main
powershell -ExecutionPolicy Bypass -File D:\Prj\gh-release-notes-skill\scripts\verify-svg-assets.ps1 -RepoPath . -Path assets/header.svg,assets/release-header-v0.1.0.svg,docs/public/release-header-v0.1.0.svg
uv run python -m compileall src scripts\mattermost_tools
uv run python -m unittest discover -s tests
uv run openclaw-podman --help

```
